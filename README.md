# Xray-CNN
من یک مدل CNN طراحی کردم که روی تصاویر X-Ray کار می‌کنه. اول داده‌ها رو نرمال‌سازی و افزایش دادم، بعد شبکه رو با چند لایه کانولوشن ساختم. این شبکه ویژگی‌های تصویر رو استخراج می‌کنه و در نهایت تصمیم می‌گیره که ریه سالمه یا بیمار. خروجی مدل هم با دقت بالای ۸۰ تا ۹۰ درصد کار می‌کنه و میشه برای پیش‌بینی روی داده‌های جدید استفاده کرد
